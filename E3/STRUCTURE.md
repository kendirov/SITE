# Структура проекта EatventureBot V3

## Корень проекта

| Файл / папка | Назначение |
|--------------|------------|
| **run.py** | Запуск бота. Точка входа. |
| **config.py** | Пороги распознавания, таймеры, зоны, пути к ассетам. |
| **requirements.txt** | Зависимости Python. |
| **README.md** | Краткое описание и быстрый старт. |
| **STRUCTURE.md** | Этот файл — схема структуры. |

## core/

Ядро бота: логика, зрение, ввод, скролл, состояние.

- **logic.py** — игровая логика (улучшения, реновация, реклама, боксы и т.д.).
- **vision.py** — скриншот и поиск шаблонов по экрану.
- **input.py** — клики, свайпы, зажатия.
- **scroll.py** — скролл вверх/вниз.
- **state.py** — состояние (счётчики, память).

## tools/

Утилиты для настройки и отладки (не основной цикл бота).

- **capture_tool.py** — снимок области игры. Сохраняет в **tools/output/**.
- **setup_zones.py** — настройка зоны игры и «опасной» зоны (бургер).
- **define_no_click_zone.py** — задание зон «не кликать».

Результаты съёмки: **tools/output/** (reference_screen_*.png).

## debug/

Отдельные скрипты для проверки фич (например, тест рекламы). Запускаются вручную при необходимости.

## assets/

Шаблоны картинок для распознавания (кнопки, иконки). Поддиректории:

- **assets/ads/** — кнопки закрытия рекламы, boost.
- **assets/No/** — картинки зон «не кликать».

## docs/

Вся документация и заметки.

- **docs/notes/** — старые заметки по фиксам, инструкции (txt/md).
- Остальные файлы в docs/ — гайды, архитектура, changelog.

## logs/

Логи бота (bot.log) — создаётся при запуске run.py. В .gitignore.

## archive/

Устаревший код и бэкапы (например, старые скрипты проверки ассетов).
