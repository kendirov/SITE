## ğŸ¯ Key Design Decisions

1. **Crop First, Detect Second**
   - Cropping to STATION_REGION before detection
   - Eliminates UI false positives at the source
   - More efficient than filtering results

2. **Click Target, Not Arrow**
   - Arrow position is reference point only
   - Actual click is offset (+20, +60)
   - Hits station counter precisely

3. **Safety Check on Target**
   - Check TARGET position, not arrow position
   - Target might be closer to danger than arrow
   - More accurate safety assessment

4. **Ignore List for Rejected Clicks**
   - Prevents spam attempts on unsafe stations
   - 20-second cooldown same as successful clicks
   - Reduces log noise

5. **Detailed Logging**
   - Shows arrow â†’ target transformation
   - Reports exact distances to danger
   - Makes debugging straightforward
